道岔 / Turnout / Points

道岔是軌道系統中的分歧結構，使列車能夠從一條軌道轉入另一條軌道。它屬於基礎設施的一部分。

分類定位 / Infrastructure Role

這一頁只聚焦軌道本體與分歧結構本身；驅動、鎖定與位置偵測設備則放在 Point Machine 頁面。

閉塞區間 / Block Sections

閉塞系統將鐵路劃分為多個可管理的區段，並利用列車偵測、號誌與行車規則維持安全列車間隔。

本頁採用簡化的固定閉塞（Fixed Block）示意模型，用於說明閉塞系統的基本概念。實際鐵路系統會依不同號誌制式、行車規則及控制系統而有所差異。

目前佔用 / Occupied Block B

在本示意模型中，當 Block B 被列車佔用時，保護該區段的 S2 顯示停止（紅燈）；前一架號誌 S1 顯示警戒（黃燈），提醒駕駛下一架號誌為停止顯示。

Note

本頁為概念示意，並非所有鐵路系統皆採用相同號誌邏輯。

聯鎖 / Interlocking

聯鎖系統負責檢查進路、道岔位置、列車偵測狀態及衝突進路，只有所有安全條件成立時，才允許建立進路並開放相關號誌。

Point Machine / 轉轍器

轉轍器是驅動、鎖定並偵測道岔位置的機電設備。聯鎖系統會根據轉轍器回報的位置與鎖定狀態，判斷是否允許建立進路，並讓相關號誌顯示允許行車（Proceed Aspect）。

列車偵測 / Train Detection

號誌系統需要先知道列車在哪裡。常見方式包含軌道電路與計軸器，兩者都用來判斷區間是否佔用。

判斷邏輯 / Detection Logic

軌道電路偵測到輪軸造成電氣分路，因此回報該區間已被佔用。

列車自動防護 / Automatic Train Protection

ATP 持續監督列車的實際速度、允許速度、煞車曲線與可行駛距離。當列車可能超過安全限制，且駕駛未及時採取動作時，系統會發出警告或介入煞車。

防護狀態 / ATP State

目前速度低於允許速度，ATP 持續監控但不介入。

列車自動監督 / Automatic Train Supervision

ATS 用於監督列車位置與運行狀態，並支援時刻表管理、列車追蹤、告警顯示與行車調度。ATS 著重營運監督，不等同於 ATP 的安全防護功能。

監督範圍 / Supervision Scope

目前先提供分類定位與系統角色說明；列車追蹤、時刻表管理與告警佇列互動將於後續擴充。

調度控制 / CTC

CTC（Centralized Traffic Control）提供集中式列車監控與行車調度功能。調度員可透過 CTC 監看列車位置、設備狀態與進路資訊，並依系統配置遠端下達控制命令。

所有涉及行車安全的控制命令仍需經由 Interlocking（聯鎖系統）進行安全條件檢查。CTC 負責監控與調度，不直接取代聯鎖的安全功能。

調度重點 / Dispatch Focus

調度員透過 CTC 集中監看列車與設備狀態；涉及行車安全的控制命令，仍需由聯鎖系統完成安全條件檢查後才會作用到現場設備。

告警與維護 / Alarm & Maintenance

維護與測試時，控制中心需要快速辨識設備異常、通訊中斷、道岔不到位或偵測異常，並判斷影響範圍。

處置重點 / Maintenance Focus
  1. 確認道岔位置偵測是否一致。
  2. 暫停相關進路建立與號誌顯示。
  3. 通知現場檢查轉轍器與鎖定機構。

測試與驗收 / Testing & Commissioning

這一頁把專案交付過程拆成 Testing、Commissioning、Acceptance 與 Handover 四個概念，並對應到從 Installation 開始的實務流程。

四個概念 / Four Concepts
  1. Testing（測試）：依照測試程序驗證設備、功能與系統是否符合設計需求。
  2. Commissioning（試運轉／啟用）：完成設備安裝與整合後，透過試運轉確認系統已具備投入營運的能力。
  3. Acceptance（驗收）：由業主依照合約及驗收標準確認工程成果是否符合要求。
  4. Handover（交付）：完成文件、設備、教育訓練及維護資料移交，正式交付業主。

平交道保護 / Level Crossing Protection

平交道保護系統通常利用列車接近資訊或列車偵測設備，依序啟動警鈴、閃光號誌及遮斷桿，以降低道路使用者與列車衝突的風險。

部分平交道會與聯鎖系統整合，只有當平交道已達到安全保護狀態時，相關進路或號誌才允許建立。

保護序列 / Protection Sequence

典型流程為：列車接近 -> 啟動警示設備 -> 遮斷桿下降 -> 列車通過 -> 解除警示 -> 遮斷桿升起。

Note

不同國家、不同路線及不同平交道型式，其控制方式可能不同。本頁內容為典型自動平交道概念示意。

ETCS / ERTMS 入門

ETCS（European Train Control System）是 ERTMS 的核心列車控制系統，用於提供列車保護與行車授權。

ETCS Level 1 / ETCS Level 2
  1. ETCS Level 1：Balise 更新資訊，地面仍保留傳統號誌
  2. ETCS Level 2：RBC 透過無線持續傳送 Movement Authority
  3. Eurobalise 在 Level 2 主要提供位置校正與參考點
